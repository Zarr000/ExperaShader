# Changelog

## v0.1.0

- Deferred/PBR foundation (GGX + Schlick Fresnel + Smith geometry)
- Deferred GBuffer passes for terrain + water
- Shadow depth pass entry shaders
- SSAO (GTAO-inspired) with blue-noise rotation
- Atmospheric scattering approximation pass
- Volumetric fog approximation pass
- Procedural volumetric cloud coverage + cloud shadow factor pass
- HDR bloom pipeline (bright-pass extract + separable Gaussian blur + combine)
- Tone mapping (ACES Filmic)
- Anti-aliasing passes: TAA and FXAA
- Basic final color controls (vignette/saturation/contrast)
